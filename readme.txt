This is Readme file of json-access Repo. Following are the main objectives of this project:
1) To provide simple, uncluttered platform for viewing and formatting json files.
2) To provide temporary storage of json files with sharable links with a certain expiry date.

This list is expected to have a few more features which will be added in due time.
Tech stack is not chosen as of yet.
jsonviewer.stack.hu is the main inspiration and will look to add onto it.
